1. O que é computação?

Computação é o estudo de processos que interagem com dados e que podem ser representados como dados na forma de programas.
A computação permite o desenvolvimento de algoritmos que são usados para processar informações, realizar cálculos e resolver problemas em diversas áreas.

2. O que é um sistema operacional?

Um sistema operacional (SO) é um software que gerencia o hardware do computador e fornece serviços para programas de aplicação.
Ele serve como uma interface entre o usuário e o hardware do computador,
facilitando a execução de programas e a gestão de recursos como memória, dispositivos de entrada e saída, e arquivos.

3. O que faz um programador?

Um programador é um profissional que escreve e mantém o código de programas de computador.
Eles desenvolvem software, escrevem código em várias linguagens de programação,
testam e depuram programas, e colaboram com outros profissionais de tecnologia para criar soluções funcionais e eficientes.

4. Primeiro site em HTML/CSS

Criar um primeiro site em HTML e CSS envolve o aprendizado dos fundamentos dessas linguagens.
HTML (HyperText Markup Language) é usado para estruturar o conteúdo da web, enquanto CSS (Cascading Style Sheets) é usado para estilizar e layoutar esse conteúdo.
Um exemplo básico de site HTML/CSS pode incluir uma página com um título, parágrafos de texto, links, e algumas formatações de cores e fontes.

5. Redes e infraestrutura

Redes de computadores consistem em um conjunto de dispositivos interconectados que compartilham recursos e informações.
A infraestrutura de rede inclui hardware (roteadores, switches, cabos) e software (protocolos,
sistemas operacionais de rede) que suportam a comunicação e a transferência de dados entre dispositivos. Redes podem ser locais (LAN) ou abrangentes (WAN).

6. Possibilidades de carreira

Na área de computação, há diversas possibilidades de carreira, incluindo, mas não se limitando a:
desenvolvedor de software, engenheiro de sistemas, analista de segurança, administrador de banco de dados, cientista de dados, engenheiro de redes, e arquiteto de soluções.
Cada uma dessas carreiras exige habilidades e conhecimentos específicos, que podem ser adquiridos através de educação formal e experiência prática.
